# CST-452 Senior Project Portfolio  
**Anaheim Ballet Website Redesign**

## Overview
This portfolio documents my Senior Project completed across CST-451 and CST-452.  
The project involved designing and developing a full-stack web application to modernize the online presence of Anaheim Ballet.

The goal of the system was to improve usability, accessibility, and engagement while following real-world software engineering practices, including documentation, testing, version control, and deployment readiness.

## Technologies Used
- **Frontend:** React, Vite, JavaScript, CSS
- **Backend:** Node.js, Express
- **Database:** MySQL (MAMP)
- **Version Control:** Git & GitHub

---

## System Features
- Responsive homepage with carousel
- Company, School, Events, Outreach, and Gallery sections
- Class calendar with level-based filtering
- Newsletter subscription (database-backed)
- Contact form with persistence
- Authentication (signup & login)
- RESTful backend API
